## 使い方
### Paste Image URL    
サムネイルの画像urlをコピーしてからこのボタンを押すと自動でurlが入力されます   
### Copy Video URL
urlを入力するところに入ってるurlを見れるように変換しその変換された動画のurlをコピーします  
### Convert and Open   
上のと違って変換しそのURLを新しいタブで開きます　　　
### Clear URL
urlを入力するところの中身を削除します
